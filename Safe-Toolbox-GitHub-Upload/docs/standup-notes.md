# Standup Notes

## Purpose
Short, session-level progress notes for Codex, Claude, and the human operator.

## Format

### YYYY-MM-DD HH:MM
- What changed:
- What was verified:
- What is still risky:
- What is next:

## Example Entry

### YYYY-MM-DD HH:MM
- What changed: Updated editor backdrop artwork and asset pipeline.
- What was verified: Build succeeded; AU refreshed; release handoff updated.
- What is still risky: Logic may still need a restart/rescan to load the newest binary.
- What is next: Host-side review of backdrop balance and control visibility.
